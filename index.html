Reactivity

Creating an object of class Vue


							new Vue ({
								el: '#app',
								data: {
									message: 'Hello, Vue!',
								},
								methods: {
									changeText () {
										this.message = 'New text'
									}
								}
							});

							<div id="app">
								<p>{{ message }}</p>
								<button v-on:click="changeText">change</button>
							</div>

  • Code Separation
  • Code separation out of sight

Directives

Directives - special attributes prefixed with v-.The directive reactively applies changes to the DOM when updating the value of this expression.


							<p v-if="seen">I can be seen now</p>

In this example, the v-if directive will remove or insert the </p> element depending on the truth of the value of the seen expression.

Animation and transitions

Vue allows you to animate transitions when adding, updating, and deleting DOM objects.

  • automatically use CSS classes for animations and transitions
  • integrate third-party libraries of CSS animations, for example, animate.css
  • use JavaScript to work with the DOM directly in transition hooks
  • integrate third-party libraries of JavaScript animations, for example, Velocity.js
